# Vibe Transfer for SD WebUI Forge

NovelAI-style vibe transfer for Stable Diffusion WebUI Forge.

## Features

- IP-Adapter semantic vibe transfer
- Reference attention + AdaIN
- Hybrid mode
- Color/lighting only mode

## Usage

1. Enable Vibe Transfer in the accordion
2. Select transfer mode
3. Upload reference image(s)
4. Adjust parameters
5. Generate

## Requirements

- Forge WebUI
- IP-Adapter models (optional, for IP-Adapter mode)
